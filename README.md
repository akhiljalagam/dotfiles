  and <code>sudo update-grub</code> 
  <br></br>
  <b>Bad TTY and GRUB resolution?</b> Add these in there:
  <pre><code>
  GRUB_GFXMODE=1920x1080x32
  GRUB_GFXPAYLOAD_LINUX=1920x1080x32
  </code></pre>
  <b>NVIDIA?</b> <code>nvidia-drm.modeset=1</code>
  <br></br>
